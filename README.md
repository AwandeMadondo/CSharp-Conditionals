# CSharp-Conditionals

This simple console app checks if a person is allowed to drive based on their age and license status.

## Features:
- Ask for name, age, and license status
- If age is under 18, they are not allowed to drive
- If age is 18 or older, it checks for license status

## Concepts Practiced:
- If/else conditions
- Nested conditionals
- User input handling
